6

自我迭代

v0.7+ meta-agent

你只说一句话。Pilot 的 meta-agent 自己读 session、定位要改的槽、提议改动、跑 dry-run、apply——你只需 approve 或 revert。

例子："以后不要用 bash 改 .ts 文件"

你说一句话 › interpret › locate slot › propose diff › dry-run eval › user approve › apply
找到 Tool Selector + Executor 阶段 → 提议 policy 改动 → dry-run 跑 N 个 session → 等 approve → 写到 Capability.lessons。出错可一键 revert。
